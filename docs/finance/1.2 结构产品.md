# 1.2 结构产品

## 收益增强型票据 Yield Enhancement Note

目的：认为市场在一段时间内横盘或微涨，期望以一定风险获得比普通存款或债券更高的收益（高于无风险利率）。

Equity Linked Note 是一种典型的 YEN。它由买入零息债券与卖出看跌期权组成。

当到期时：

- $S\ge K$：期权买家不行权，你取得本金的无风险利率增值 + 卖出期权的权利金。
- $S\lt K$：期权买家行权，你使用本金用高价买入贬值的股票（接货）。

通常确保本金在到期应等于卖出期权时的股票价值（$Ve^{rt} = \Delta K$），这样当接货时，本金总能覆盖这部分费用。如果本金不能完全覆盖股票价值，之后就是杠杆的范围了。

卖出看涨期权也是同样的道理，但如果触发了接货，那手中连贬值的股票也没有了，因此相比 ELN 更不常见。

## 保本票据 Capital Guaranteed Note

目的：牺牲无风险利息，博取潜在更高的收益。

它由买入零息债券与买入看涨期权组成，并使零息债券在到期时能补足本金的数额。

当到期时：

- $S\gt K$：行权获利，你取得本金 + 行权收益。
- $S\le K$：期权无价值，你取得由债券补足的全部本金，但没有额外增长，等于期间没有无风险利息收益。
