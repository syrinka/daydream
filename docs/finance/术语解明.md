# 术语解明

## 期货 Futures

由对等双方签订的对称协约，规定在未来的约定时间买卖双方可以、且必须以既定价格进行交易。

通常会采取保证金（Margin）形式确保协约履行。

## 期权 Options

非对称协约，规定在未来的约定时间买方可以、非必须以既定价格进行交易，此行为也被称为行权（Exercise）。

出于风险的不对称，买方需支付卖方额外的权利金（Premium）。

## 头寸 Position

头寸描述持有的资产（股票、债券、期权等）的**数量及方向**。

- 多头头寸（Long Position）：此资产（或它的标的资产）在价格上涨时你可盈利
- 空头头寸（Short Position）：此资产（或它的标的资产）在价格下跌时你可盈利

## 投资组合 Portfolio

所持有的所有头寸的集合。
